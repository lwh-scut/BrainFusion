BrainFusion.viewer.plot module

Module for plotting EEG raw data and power spectral density topography.

Provides functions to visualize EEG time series and frequency distributions.

BrainFusion.viewer.plot.plot_eeg_psd(data, is_relative=False, is_norm=True, title='')

Visualize EEG power spectral density (PSD) using topographic maps.

Parameters:

  • data (dict) EEG data dictionary with signals and metadata

  • is_relative (bool) Flag for relative power normalization

  • is_norm (bool) Flag for min-max normalization

  • title (str) Plot title

BrainFusion.viewer.plot.plot_raw(data, channel=None, sharey=False, line_color='black', linewidth=0.5, is_save=False, save_path=None)

Visualize EEG time series data for single or multiple channels.

Parameters:

  • data (numpy.ndarray) EEG time series data (1D vector or 2D array)

  • channel (list[str], optional) List of channel names

  • sharey (bool) Share Y-axis scales across channels

  • line_color (str) Color for signal traces

  • linewidth (float) Width of signal traces

  • is_save (bool) Flag to save the plot

  • save_path (str, optional) Output path for saving the plot
