Creates a standard BDF file from provided data

Parameters:

  • file_name (str) – Output filename

  • n_channels (int) – Number of channels

  • signals (np.ndarray) – Signal data (n_channels x n_samples)

  • channel_names (list[str]) – List of channel names

  • physical_mins (list[float], optional) – Physical minimum values, defaults to [-10000]*n_channels

  • physical_maxs (list[float], optional) – Physical maximum values, defaults to [10000]*n_channels

  • digital_mins (list[int], optional) – Digital minimum values, defaults to [-32768]*n_channels

  • digital_maxs (list[int], optional) – Digital maximum values, defaults to [32767]*n_channels

  • sampling_frequency (float, optional) – Sampling rate, defaults to 256

  • technician (str, optional) – Technician name, defaults to ‘’

  • equipment (str, optional) – Equipment description, defaults to ‘’

  • patient_code (str, optional) – Patient code, defaults to ‘’

  • patient_name (str, optional) – Patient name, defaults to ‘’

  • birthdate (str, optional) – Patient birthdate, defaults to None

  • admin_code (str, optional) – Administration code, defaults to ‘’

  • annotations (list[tuple], optional) – Event annotations [(onset, duration, description)], defaults to None

  • transducers (list[str], optional) – Transducer types, defaults to [‘EEG Electrode’]*n_channels

  • physical_dimensions (list[str], optional) – Physical dimensions, defaults to [‘uV’]*n_channels
